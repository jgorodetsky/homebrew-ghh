# homebrew-ghh

Homebrew tap for [ghh](https://github.com/jgorodetsky/ghh) â€” your entire git + GitHub workflow in one searchable command.

## Install

```bash
brew tap jgorodetsky/ghh
brew install ghh
```

Then add this to your `~/.zshrc` or `~/.bashrc`:

```bash
source $(brew --prefix)/bin/ghh
```

Restart your shell and type `ghh`.

## Upgrade

```bash
brew upgrade ghh
```

## Uninstall

```bash
brew uninstall ghh
brew untap jgorodetsky/ghh
```
